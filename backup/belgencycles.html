Belgen Cycles

(Web Design Project)

The intent of this was to redesign/remake a website for Belgen Cycles/Village Bike in Richmond, Vermont. To create a harmonious design using a great layout and to strengthen our skills for SkillsUSA Nationals.

Remaking Belgen Cycles website to a more modern, stylistic version. I wanted to make a sectioned style website with background images and content in the center to fill with text. A slideshow created using JavaScript was made in the bikes tab to show off the various bikes up for sale including the name brand.

I started creating the different sections on the front page to show off what to say. For the background images, I specifically picked ones that used leading lines to create a nice focal point on the bike. The image I chose for the header used asymmetry to give off visual hierarchy and to use contrast well for the text. I made all the images on the website fixed (stay still on the webpage) to make all the content on each section look very professional.
